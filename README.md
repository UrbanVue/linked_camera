# Linked Camera

**Linked Camera** is a lightweight, privacy-aware Android camera app designed for field data collection.

Built upon [Open Camera](https://opencamera.org.uk/) by Mark Harman.

## Quick Start

### Nextcloud Setup

1. Create a Nextcloud public share with upload permissions
2. Settings ? Linked Camera Settings ? Server settings
3. Enable Nextcloud Auto-Upload and paste share URL

For full documentation, see releases page.
